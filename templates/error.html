Status: Configuration Needed



To begin, create a file in the same directory as this binary.

Name it conf and place each player's name in the file, separated by a new line. Eg:


Josh
Sam
Brian
Ralph

Restart the program to initialize a survivor game.

Once a game is started, *.survive files will be created in the same directory you ran this binary from.

To maintain persistence, always run this binary with the accompanying .survive and conf files. Don't modify these files directly.
